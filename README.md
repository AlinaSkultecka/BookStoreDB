# Bookstore SQL Database Project

**Description:**  
A relational SQL Server database for a multi-shop bookstore, supporting books, authors, shops, publishers, customers, orders, and stock tracking.

**Key Features:**
- Tables for Authors, Books, Shops, Publishers, Stock, Customers, Members, Orders
- Contains sample data with Russian literature and clients (in English)
- `MoveBook` stored procedure for transferring stock safely between shops
